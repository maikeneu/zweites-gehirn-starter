# Daily Notes

Hier liegt dein Tagebuch der Zusammenarbeit. Eine Notiz pro Tag, benannt nach dem Datum im Format JJJJ-MM-TT, zum Beispiel 2026-06-19.

So sortieren sich die Tage von allein chronologisch. Dein Assistent kann hier nachlesen, was zuletzt passiert ist, und behält den roten Faden zwischen euren Sitzungen.

Du musst das nicht täglich führen. Schon ein kurzer Eintrag an arbeitsreichen Tagen hilft.
